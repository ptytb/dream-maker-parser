#DreamMaker programming language parser.
[Byond](http://www.byond.com) is game engine using DreamMaker as native
programming language.

#Requirements
- ANTLR 4.2;
- JDK 8.

#Notes
##Building AST
Running parser requires java **-Xmx512M** command line key.
